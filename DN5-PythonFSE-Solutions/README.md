# Cognizant DN 5.0 Python FSE - Database Integration

This repository contains the solutions for **Module 3: Database Integration** of the Cognizant Digital Nurture 5.0 Python Full Stack Engineer Track.

## Directory Structure

- `Module3_DatabaseIntegration/vitta_hem_kumar/` - Main solutions directory for Vitta Hem Kumar.
  - SQL Hands-on exercises (`hands_on_1.sql` to `hands_on_4.sql`).
  - `mongodb/` - MongoDB JS exercise solutions.
  - `orm/` - Python ORM (SQLAlchemy, Flask-SQLAlchemy, Alembic) solutions, dependencies, and database utilities.
  - `migrations/` - Database migrations documentation and configuration.
  - `screenshots/` - Execution evidence screenshots organized by exercise.
