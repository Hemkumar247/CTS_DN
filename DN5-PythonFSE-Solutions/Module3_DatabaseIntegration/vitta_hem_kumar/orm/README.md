# ORM Exercises with SQLAlchemy & Flask-SQLAlchemy

This directory contains database solutions built using Object-Relational Mapping (ORM) technologies.

## Files

- `database.py` - Database session initialization, engine configuration, and Declarative Base.
- `models.py` - Database tables represented as python model classes.
- `crud.py` - CRUD utilities helper functions to interact with the database.
- `requirements.txt` - Python dependency requirements.
- `.env.example` - Template config file for environment variables.
